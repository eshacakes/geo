GEO ELEMENT

One of the most straightforward elements in this game is the Geo element. It provides shields and protections. Geo can also work like Anemo, but a bit more niche. This element can break shields quickly without needing a specific element. Any Geo reaction creates a crystal of the element it came into contact with and grants a shield once it is picked up.

Geo Element Characters

All Cryo characters in Genshin Impact are Noelle, Ningguang, Gorou, Yun Jin, Albedo, Itto, and Zhongli.
